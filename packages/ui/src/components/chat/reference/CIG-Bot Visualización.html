Tarjetas en esta conversación

ALT
Panel de alertas
Semáforo con barra de estado, íconos por módulo y conteo. Siempre es el primer mensaje del día.
SPK
Ventas últimos 7 días
KPIs resumen + sparkline total + mini-tendencias por canal (Online, Presencial, B2B).
GAU
Meta del mes
Gauge donut con % de cumplimiento. Color cambia por umbral: verde ≥80%, ámbar 50–79%, rojo <50%.
TRZ
Trazabilidad de bloqueo
Timeline causal con cada paso del motor de reglas. Cita la política exacta (CAR-04 §2.3).

Sistema de calificación

★
Escala 1 – 5 estrellas
Cada tarjeta tiene su propia calificación independiente. El hover ilumina de forma acumulativa.
LOG
KPI Logger
Guarda en kpi_logs.db: interaccion_id, intención, tarjeta y calificación (1–5).
LLM
Mejora del modelo
Calificaciones ≤ 2 acumuladas por intención disparan revisión del prompt del LLM.

Fuentes de datos

ERP
Módulos conectados
Inventario, Ventas, Metas, Crédito y motor de reglas (CAR-04). Conector FastAPI.
KMS
Políticas internas
Base documental indexada. Cada respuesta cita sección exacta del documento fuente.
