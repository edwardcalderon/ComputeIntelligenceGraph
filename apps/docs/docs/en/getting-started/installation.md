# Installation

## Prerequisites

- Node.js 20.0 or higher
- pnpm 9.0 or higher

## Installation Steps

1. Clone the repository
2. Install dependencies
3. Configure environment variables
4. Start the development server

For detailed instructions, see the [Quick Start](./quick-start.md) guide.
