# Getting Started

Welcome to the CIG (Compute Intelligence Graph) documentation. This section helps you get up and running with the current product state: live or demo graph exploration, chat-assisted operations, and managed or self-hosted installs.

## What is CIG?

CIG is a platform for discovering infrastructure, indexing it into a graph, querying it with Cypher and chat, and operating it through the dashboard or CLI.

## Current Surfaces

- Landing: `https://cig.lat`
- Dashboard: `https://app.cig.lat`
- API: `https://api.cig.technology`
- Docs: `https://cig.lat/documentation`

## Operating Modes

- `live` uses real discovery-backed infrastructure.
- `demo` uses the shared seeded demo workspace and is useful when live discovery is unavailable.
