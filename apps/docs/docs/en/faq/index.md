# Frequently Asked Questions

## General Questions

### What is CIG?

CIG (Compute Intelligence Graph) is a platform for managing and analyzing compute intelligence graphs.

### Who should use CIG?

CIG is designed for:
- Data scientists
- Software engineers
- System architects
- Anyone working with graph data

### Is CIG open source?

Yes, CIG is open source and available on GitHub.

## Technical Questions

### What are the system requirements?

- Node.js 20.0 or higher
- pnpm 9.0 or higher
- 4GB RAM minimum
- 10GB disk space

### How do I get started?

See the [Quick Start](../getting-started/quick-start.md) guide.

### How do I report a bug?

Create an issue on [GitHub](https://github.com/edwardcalderon/ComputeIntelligenceGraph/issues).

## Support

### Where can I get help?

- Check the [Troubleshooting](../troubleshooting/index.md) guide
- Search existing GitHub issues
- Create a new GitHub issue
- Contact the development team
